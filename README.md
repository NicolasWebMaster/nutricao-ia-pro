# nutricao-ia-pro
Chatbot inteligente de nutrição baseado em IA, RAG e evidências científicas. Utiliza LangChain, FAISS, embeddings semânticos e Groq LLM para responder perguntas com base em fontes confiáveis como OMS, NIH, CDC, Harvard e Ministério da Saúde, reduzindo alucinações e aumentando precisão contextual.
